Q:有关只希望导入部分材质/音效
A:解压后（反正你能看到这个文档就说明已经解压了）把asset下对应文件大概删一下就OK了
比如如果你觉得音效太吵了可以把assets/minecraft/sounds删了
比如你只想要床的材质就可以把assets内除了textures/entity/bed以外的所有文件都删了
其他的大家自己研究一下吧
还有就是assets外面的pack.mcmeta一定不要删！删了可能会导致MC认不出来这是个材质包
